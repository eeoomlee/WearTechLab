# Soo-Min Lee Homepage

This is a lightweight static homepage designed for GitHub Pages.

## Files

- `index.html` — website content
- `style.css` — visual design
- `script.js` — mobile menu and year update
- `CNAME` — custom domain placeholder

## How to edit

Open `index.html` in a text editor and replace sample publications, projects, and lab information.

## How to publish on GitHub Pages

1. Create a GitHub repository named something like `soominlee-homepage`.
2. Upload these files to the repository root.
3. Go to repository **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. GitHub will provide a free URL like:
   `https://yourgithubusername.github.io/soominlee-homepage/`

## Custom domain

Replace the content of `CNAME` with your real domain, for example:

```txt
www.yourdomain.com
```

Then configure DNS at your domain registrar.
